# 用户增长分析项目集

本仓库收录了两个与电商平台用户增长相关的基础分析项目，主要用于展示对用户增长框架、漏斗分析、留存分析以及结构化业务分析能力的理解。

## 项目目录

### 1. Shopee 漏斗分析项目
基于 AARRR 模型与漏斗分析，对电商平台用户从访问到下单的行为路径进行分析。  
重点包括：
- AARRR 模型
- 漏斗分析
- 转化率分析
- 流失环节识别
- A/B Test 思路

### 2. Shopee 留存分析项目
围绕用户首次进入平台后的回访情况，分析用户是否会持续回到产品中使用。  
重点包括：
- Day 1 / Day 7 / Day 30 留存分析
- 用户粘性分析
- 留存问题识别
- 召回策略思路
- 长期用户价值

## 展示能力
- 用户增长分析思维
- 数据解释能力
- 结构化业务分析能力
- 增长建议输出能力
- 基础产品分析能力
