# Shopee 漏斗分析数据表

## 原始数据表

| 阶段 | 用户数 |
|---|---:|
| 访问 | 1000 |
| 注册 | 600 |
| 浏览 | 500 |
| 加购 | 200 |
| 下单 | 120 |

## 转化率表

| 阶段 | 转化率 |
|---|---:|
| 访问 → 注册 | 60.0% |
| 注册 → 浏览 | 83.3% |
| 浏览 → 加购 | 40.0% |
| 加购 → 下单 | 60.0% |

## 结论
- 最大流失环节出现在“浏览 → 加购”
- 该阶段转化率仅为 40.0%
- 说明用户对商品产生兴趣，但没有被充分说服进行下一步购买行为
